## Setup

To run the sample code, you need to add a `config.py` file in the root directory of the sample code. The `config.py` file should contain the following code:

```python
config = {
    "username": "username",
    "password": "password",
    "serviceApiUrl": "serviceApiUrl",
    "apiType": "admin",
    "debugMode": False
}
```
